# PSG Visitors Management - Frontend

A modern, responsive web application frontend for the PSG Visitors Management built with React, Vite, and Tailwind CSS.

## 🎯 Features

## 🛠️ Tech Stack

- **Frontend:** React 19.2.0, Vite 7.2.2, Tailwind CSS 4.1.17
- **Animations:** Framer Motion 12.23.24
- **Icons:** Lucide React 0.554.0
- **Routing:** React Router DOM 7.9.6
- **Deployment:** Docker + Nginx (SPA routing)
